V3.0.01 - 01.10.2020

Dashboard
  • ToDo-Liste
    • Die ToDo-Liste findet sich jetzt rechts oben. Durch Klicken auf das Symbol, werden die ToDos angezeigt.
  • Geburtstage
    • Wenn ein Assistent oder mehrere Assistenten in einem Monat Geburtstag haben, wird neben dem Symbol für ToDos ein Geburtstagskuchen-Icon eingeblendet. Mit Klick auf den das Icon kann man sehen wer in diesem Monat, an welchem Tag Geburtstag hat.
    • Das Icon taucht nicht auf, wenn in einem Monat niemand Geburtstag hat
    • Geburtstage von Assistenten mit dem Status "Inaktiv" werden nicht berücksichtigt
  • Die Bereiche "öffentliche Projekte", "Industrieprojekte", "Abschlussarbeiten" und "Veröffentlichungen" greifen nun auf unterschiedliche Dateien zu. Der HTML-Kenner kann sich so sein eigenes Layout schnell zusammenbauen, z.B. die Reihenfolge ändern :)
  • Alle Einträge, bei denen es geht, werden jetzt nach Datum sortiert
  • In "Industrieprojekte" und "öffentliche Projekte" werden jetzt nur noch die Kurztitel genannt, um den Platz besser auszunutzen
